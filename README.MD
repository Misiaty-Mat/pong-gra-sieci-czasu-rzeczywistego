# Wymagane narzędzia:
- Node.js
- NPM (Node Package Manager)

# Uruchomienie projektu:
### Komendy do wykonania w bazowym katalogu projektu
- `npm install` 
- `npm start`

# Jak używać aplikacji lokalnie?
- Domyślny adres `http://localhost:3000/`
- By uruchomić rozgrywkę lokalnie należy otworzyć drugą zakładkę i również wejść na podany adres
